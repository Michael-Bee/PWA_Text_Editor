## <span id="description">Description:</span>
An installable text editor application that takes notes with JavaScript syntax highlighting.

## <span id="installation">Installation:</span>
1. Clone the remote repository from GitHub
2. Install dependencies: `nmp install`
3. Initiate program: `npm run start`

The app can also be installed from the browser window.


## <span id="usage">Usage:</span> 
This application can be used to write text or basic code and saved.
